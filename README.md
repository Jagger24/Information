# How To Use Github + Command Line

## Pulling From Github

Use the following command:

```bash
git fetch
git merge
```

## Pushing To Github

Use the following command:
```bash
git add .
git commit -m "Insert comment here"
git push
```

Add your Github username and password

## Other helpful hints for command line

* List files
```bash
ls
```
* Change directory
```bash
cd DirectoryName
```
